🚀 Git & GitHub Setup Guide

✅ Try these steps to set up your GitHub repository:

1. Navigate to your project directory
   Open your terminal and go to your project folder:
   > cd path/to/your/project

2. Initialize a new Git repository
   This sets up Git tracking in your project:
   > git init

3. Link your local repository to GitHub
   Replace the URL with your actual GitHub repository link:
   > git remote add origin https://github.com/yourusername/your-repo-name.git

📁 How to Add Files to Your Repository

1. Check the status
   See which files are untracked or modified:
   > git

2. Add the files to be committed
   Add all files or specify individual files:
   > git add .
   or
   > git add filename

3. Commit your changes
   Save your changes with a descriptive message:
   > git commit -m "Initial commit"

4. Push changes to GitHub
   Upload your local commits to the GitHub repository:
   > git push -u origin master

5. Pull changes from GitHub
   Sync your local repository with the remote one:
   > git pull origin master


: Installation & Configuration

Installation is straightforward via git-scm.com . However, the "One-Time Setup" is critical. Git needs to know who you are.
You must run:

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

Important Note: Use the same email address here that you use for your GitHub account. This links your local commits to your online profile so you get credit (the green squares!) on your contribution graph.
